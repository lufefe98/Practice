Standard CSS Box Model

This div uses the standard box model

The margin, padding, border and content all add up to give a cumulative distance along with the height, vertically and width horizontally

The amount of space the box takes up will be a combination of the width/height (600px or 400px, respectively), the padding on either side of the content (50px twice) and the border on either side (10px) which add up to a height of 520px and a width of 720px

Alternative CSS Box Model

This div uses the Alternative box model

Because the box-sizing of this div was set to border-box it follows the alternative box-model where the size of the box is strictly limited to the width and height that is set for it.

The space of the box is the height (400px), vertically and the width (600px), horizontally. For the size of the content, it is the height or width minus the padding and border, unlike the standard model where the content, the padding and the border all add up and increase the box size.
