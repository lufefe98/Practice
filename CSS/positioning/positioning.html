Static Box

Can't use top, left, right nor bottom properties

Relative Box

Positioned according to normal document flow and offset relative to itself based on top/bottom/left/right Top: 230px

Absolute Box

Removed from normal flow of the document and has no space on the created for it in page layout. It's positioned relative to nearest ancestor. Otherwsie placed relative to nearest containing block. The final position is based on top/bottom/left/right value. Top: 230px; left: 230px

Fixed Box

Removed from normal document flow. Positioned relative to initial containing block determined by viewport (unless ancestor has filter/perspective/transform or will-change set to transform) Final position based on top/bottom/left/right value. Top: 250px

Sticky Box

Part of normal document flow, then offset relative to nearest scrolling ancestor and and containing block (nearest block-level element) including table related elements based on top/bottom/left/right value. The offset won't affect other elements

Absolute box is either going to be positioned relative to its nearest non-static ancestor or the initial contianing block. (THIS DIV IS IN A STATIC CONTAINER)
